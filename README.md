# 🚀 Modern Glassmorphism Portfolio

A sleek, high-performance personal portfolio website featuring a modern **Glassmorphism** and **Cyberpunk-inspired** dark aesthetic. This project showcases professional experience, technical skills, and featured projects with a focus on smooth animations and responsive design.

## ✨ Key Features

  * **Theme Toggle**: Seamless switching between **Dark Mode** and **Light Mode**.
  * **Interactive Typing Effect**: Engaging hero section displaying roles like Full Stack Developer and MERN Engineer.
  * **Glassmorphism Design**: Use of backdrop filters, subtle grain textures, and neon glow effects for a premium feel.
  * **Fully Responsive**: Optimized for all devices using a mobile-first approach and CSS Grid/Flexbox.
  * **Smooth Navigation**: Integrated Intersection Observer for fade-up scroll animations and active link highlighting.
  * **Functional Contact Form**: Connected to Google Apps Script for real-time message handling.
